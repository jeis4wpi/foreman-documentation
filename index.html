The docs.theforeman.org documentation site is work in progress. Some guides are not yet available for all flavors and some content might be missing. See related GitHub milestone for details on our progress.

For official documentation, see Foreman Manual. For Foreman plugin overview, see Plugins.

Navigating the docs.theforeman.org site

Use the top menu bar to select your release version and flavor.

For questions, visit the support forum. For contribution, visit the GitHub repository.

Available flavors

We provide documentation for the following flavors:

  • Foreman on Enterprise Linux (EL)

  • Foreman on Debian/Ubuntu

  • Foreman with the Katello plugin on Enterprise Linux (EL)

Supported releases

The following releases are supported and receive updates, including security updates:

  • Foreman 3.14 - Katello 4.16

  • Foreman 3.13 - Katello 4.15

Important

If you are currently using one of the unsupported earlier releases, we recommend you to update to a supported release.
